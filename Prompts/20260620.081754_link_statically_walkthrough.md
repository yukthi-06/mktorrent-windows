## Decisions made during implementation
- Added `LDFLAGS="-static"` to the `make` build command in the workflow. This instructs the linker to statically bundle Windows runtime dependencies (such as `libwinpthread-1.dll` and `libgcc_s_seh-1.dll`) into the output `mktorrent.exe` executable, allowing it to run standalone without runtime DLL dependencies.
